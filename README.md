## Run Script

First give permission to the script:

```shell
chmod +x script.sh
```

Then run it:

```shell
./automationConfig.sh
```

## Tmux

- Path: `~/.tmux.conf`

## Bashrc

- Path: `~/.bashrc`

## Alacritty

- Path: `~/.config/alacritty/alacritty.toml`

## VSCode

- Path: any place will be fine
